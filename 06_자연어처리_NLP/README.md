# 자연어처리 (NLP) 학습 로드맵

이 폴더는 자연어처리(Natural Language Processing)의 기초부터 심화까지 단계별로 학습할 수 있는 자료들을 포함하고 있습니다.

## 📂 폴더 구조

### 01_text_analysis
텍스트 분석의 기초를 다룹니다.
- `01_extract_keyword_frequency.ipynb`: 텍스트에서 키워드 빈도 추출
- `02_wordcloud.ipynb`: 워드클라우드 시각화
- `03_network_visualization.ipynb`: 단어 간 네트워크 시각화

### 02_preprocessing
텍스트 전처리 기법을 학습합니다.
- `01_tokenization.ipynb`: 토큰화 (Tokenization)
- `02_pos_tagging.ipynb`: 품사 태깅 (POS Tagging)
- `03_stopword_removal.ipynb`: 불용어 제거

### 03_text_vectorization
텍스트를 숫자로 변환하는 벡터화 기법입니다.
- `01_one_hot_encoding.ipynb`: 원-핫 인코딩
- `02_count_vectorizer.ipynb`: 카운트 기반 벡터화
- `03_tfidf_vectorizer.ipynb`: TF-IDF
- `04_word2vec.ipynb`: Word2Vec 워드 임베딩

### 04_sequence_model
시퀀스 데이터 처리를 위한 딥러닝 모델입니다.
- `01_rnn.ipynb`: 순환 신경망 (RNN) 기초
- `02_lstm.ipynb`: LSTM
- `03_gru.ipynb`: GRU

### 05_text_classification
텍스트 분류 실습입니다.
- `01_naive_bayes_classifier.ipynb`: 나이브 베이즈 분류기
- `02_rnn_classifier.ipynb`: RNN 기반 분류
- `03_multi_label_classification.ipynb`: 다중 레이블 분류

---
