# 멀티모달 (MultiModal) 학습 로드맵

이미지와 텍스트를 함께 처리하는 멀티모달 딥러닝 기술을 학습합니다. CNN 기초부터 심화까지 다룹니다.

## 📂 폴더 구조

### 01_cnn
CNN(Convolutional Neural Network)의 기본 구조와 활용을 다룹니다.
- `01_cnn.ipynb`: CNN 모델 구현 및 이미지 분류 실습

### study
CNN의 핵심 구성 요소와 심화 개념을 깊이 있게 학습합니다.
- `01_padding_stride`: 패딩(Padding)과 스트라이드(Stride)의 이해
- `02_pooling`: 풀링(Pooling) 레이어의 역할과 종류
- `03_cifar10`: CIFAR-10 데이터셋을 활용한 이미지 분류 심화 실습

---
*추후 VQA, Image Captioning, Diffusion Model 등 다양한 멀티모달 주제가 추가될 예정입니다.*
