# 🛠️ 00. 환경설정: AI 요리를 위한 '주방 공사'

요리(코딩)를 하려면 먼저 최신식 주방(개발 환경)이 필요합니다.
이 **3가지 도구**만 깔면, 당신은 전 세계 AI 개발자와 똑같은 환경을 갖추게 됩니다.

---

## 1. 아나콘다 (Anaconda): 재료 창고 🥦
파이썬과 필수 라이브러리(Numpy, Pandas 등)를 **세트 메뉴**로 설치해주는 도구입니다.
- **비유**: 텅 빈 냉장고(Python만 설치) vs **꽉 찬 식재료 창고(Anaconda)**.
- **설치**: [Anaconda 공식 홈페이지](https://www.anaconda.com/) -> Download -> 설치 파일 실행.

## 2. VS Code (Visual Studio Code): 요리 도구 🔪
코드를 작성하고 실행하는 **편집기(Editor)**입니다.
- **비유**: 메모장(과도) vs **VS Code(전문 셰프용 칼)**. 자동 완성 기능이 강력합니다.
- **설치**: [VS Code 공식 홈페이지](https://code.visualstudio.com/) -> Download.
- **필수 확장팩**: VS Code 실행 후 왼쪽 '블록' 아이콘 클릭 -> `Python`, `Jupyter` 검색해서 설치.

## 3. 깃 (Git): 레시피 기록장 📝
코드의 변경 내역을 저장하고 관리하는 **타임머신**입니다.
- **비유**: 실수로 소금 왕창 넣었을 때, **넣기 전으로 시간을 되돌리는 기능(Commit & Reset)**.
- **설치**: [Git 공식 홈페이지](https://git-scm.com/).
- **사용법**:
  ```bash
  git init        # 기록장 만들기
  git add .       # 변경된 파일 찜하기
  git commit -m "첫 번째 코드 저장"  # 기록장(저장소)에 저장
  ```

2. **VS Code**로 요리(코딩)를 하고,
3. **Git**으로 레시피를 기록한다.
