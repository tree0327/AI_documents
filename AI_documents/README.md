# 🗺️ 누구나 쉽게 배우는 AI 로드맵 (AI Map)

안녕하세요! 이 폴더는 **AI와 프로그래밍을 처음 접하는 분**들을 위해 설계되었습니다.
어려운 수학 공식 대신, **친숙한 비유(Metaphor)**를 통해 개념을 먼저 잡고 실습하는 방식입니다.

## 📂 폴더 구조 및 학습 가이드

### **Phase 1: 기초 다지기 (Foundation)**
- **📁 01_파이썬_기초**: AI의 언어인 파이썬을 배웁니다.
    - `01_파이썬_시작하기.ipynb`: **파이썬 = 통역사**. 컴퓨터와 대화하는 법.
    - `02_파이썬_중급_붕어빵.ipynb`: **클래스 = 붕어빵 틀**. 더 효율적으로 코딩하기.
    - `03_파이썬_고급_선물포장.ipynb`: **데코레이터 = 선물 포장**. 고급 문법 꿀팁.

- **📁 02_데이터_정제**: 재료(데이터)를 구하고 다듬습니다.
    - `01_웹크롤링/01_크롤링_기초_배달부.ipynb`: **Requests = 배달부, BeautifulSoup = 번역가**. 인터넷에서 보물 찾기.
    - `02_MySQL`: **DB = 거대한 도서관**. 데이터를 체계적으로 저장하기.

- **📁 03_데이터_분석**: 데이터를 요리하고 맛봅니다.
    - `01_넘파이_기초_컨테이너.ipynb`: **List = 배낭 vs Array = 컨테이너 트럭**. Numpy가 빠른 이유.
    - `02_판다스_엑셀.ipynb`: **Pandas = 엑셀의 파이썬 버전**. 표 데이터 다루기.
    - `03_시각화_그림그리기.ipynb`: **시각화 = 플레이팅**. 데이터를 예쁜 그래프로 그리기.

---

### **Phase 2: 머신러닝 & 딥러닝 (Core AI)**
- **📁 04_머신러닝**: 컴퓨터에게 경험을 가르칩니다.
    - `01_의사결정나무/01_의사결정나무_스무고개.ipynb`: **의사결정 나무 = 스무고개 게임**. 질문을 통해 정답 찾기.
    - `02_KMeans`: **군집화 = 끼리끼리 모으기**. 비슷한 데이터끼리 그룹 짓기.

- **📁 05_딥러닝**: 인간의 뇌를 흉내 냅니다.
    - `01_경사하강법_산내려오기.ipynb`: **경사하강법 = 눈 가리고 산 내려오기**. 발끝의 감각으로 최적의 값 찾기.
    - `02_MLP`: **신경망 = 뉴런의 연결**. 뇌세포처럼 학습하기.

---

### **Phase 3: 언어 모델의 혁명 (NLP & LLM)**
- **📁 06_자연어처리**: 컴퓨터가 글을 읽는 법.
    - `00_전처리_재료손질.ipynb`: **전처리 = 재료 손질**. 글자를 다듬는 법(토큰화).
    - `01_벡터화_메뉴판.ipynb`: **원-핫 인코딩 = 메뉴판 도장 찍기**. 글자를 숫자로 변환.
    - `03_임베딩_단어지도.ipynb`: **임베딩 = 단어 지도 만들기**. 비슷한 단어는 가까이.

- **📁 07_거대언어모델_LLM**: 요즘 핫한 거대 언어 모델.
    - `01_트랜스포머_동시통역.ipynb`: **RNN = 이어달리기 vs 트랜스포머 = 동시 통역**. 혁명의 시작.
    - `02_프롬프트_지니의램프.ipynb`: **프롬프트 = 지니의 램프**. 소원을 구체적으로 비는 법.
    - `03_RAG_오픈북시험.ipynb`: **RAG = 오픈 북 테스트**. AI에게 참고서를 쥐어주기.

---

### **Phase 4: 실전 응용 (Application & Project)**
- **📁 08_스트림릿**: 나만의 웹 서비스 만들기.
    - `감성분석기_앱.py`: **AI 감성 분석기**. 파이썬만으로 뚝딱 만드는 웹 앱.

- **📁 09_종합프로젝트_주식비서** (New! 👑): **배운 모든 기술 총집합!**
    - `01_데이터수집_준비.ipynb`: 주가 데이터 수집 (크롤링/API).
    - `02_모델학습_LSTM.ipynb`: 미래 주가를 예측하는 AI 만들기 (딥러닝).
    - `03_실전앱_배포.py`: 내 주식을 예측해주는 웹 앱 만들기 (스트림릿).
